A la hora de crear cualquier tipo de proyecto cuyo objetivo principal es su ejecución en la web, es indispensable una planificación previa basada en investigaciones donde se determina las necesidades, deseos, expectativas y objetivos de los usuarios finales para quienes se está diseñando el producto.
A través de la experiencia de usuario es posible captar, entender y atender la percepción y emoción general que una persona tiene al interactuar con un producto, servicio o sistema digital, abarcando desde la facilidad de uso, eficiencia y accesibilidad, hasta la satisfacción y el placer al lograr sus objetivos.



Es por eso que, para lograr lo anterior, se puede implementar la maquetación de sitios web. Es el proceso de estructurar y organizar visual y funcionalmente los elementos de una página (textos, imágenes, botones, menús) usando lenguajes como HTML y CSS para transformar un diseño estático en código que los navegadores entienden, asegurando que sea atractiva, fácil de usar, accesible y adaptable a diferentes dispositivos (móviles, tablets, PCs).
Con la maquetación la meta es crear una experiencia de usuario (UX) fluida, rápida y agradable, más allá de la estética, priorizando la usabilidad y el rendimiento. Además de asegurar que la web se vea y funcione bien en cualquier tamaño de pantalla, desde un smartphone hasta un monitor grande.
